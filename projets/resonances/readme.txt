Jours 26 - 16 au 19 janvier 2017 : Challenge Landing page (intégration d'une maquette .psd)

1) Analyse de la maquette
2) Définition sur papier des différentes sections (balise à utiliser)
3) Developpement du HTML puis validation du W3C
4) Mise en forme via le CSS

Difficultés rencontrés :
- Le positionnement (float, ect...) (Actualités)
- Le dégradé sur le background (header)

Apprécié :
- Au delàs de la mise en forme : L'animation des liens (hover)

Remarques de l'intervenant :
- transitions sur les éléments de base et non sur les hovers
- background sur les chevrons (Clins d'oeil)
- Fonction column sur le paragraphe Présentation
- Désactiver le resize du textarea dans le formulaire
- Utiliser le float pour les formulaires
